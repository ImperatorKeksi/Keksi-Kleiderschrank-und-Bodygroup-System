# Entity Icon für Wardrobe Dresser

Da ich kein Bild erstellen kann, musst du eine 64x64 Pixel große PNG-Datei erstellen:

**Dateiname:** wardrobe_dresser.png
**Pfad:** materials/entities/wardrobe_dresser.png
**Größe:** 64x64 Pixel
**Format:** PNG mit transparentem Hintergrund

**Inhalt:** Ein kleines Icon vom Dresser oder ein Kleiderschrank-Symbol

Du kannst:
1. Ein Screenshot vom Dresser-Model machen und auf 64x64 verkleinern
2. Ein eigenes Kleiderschrank-Icon zeichnen
3. Ein kostenloses Icon aus dem Internet verwenden (z.B. FlatIcon, mit entsprechender Lizenz)

Alternativ funktioniert die Entity auch ohne Icon, dann wird ein Standard-Placeholder verwendet.
